IMPLEMENTAÇÃO NECESSÁRIA:

1. CRIAR RELAÇÃO BANCO ↔ TIPOS DE CONTRATO:
   - Verificar de onde vem a lista de tipos de contrato
   - Provavelmente vem da tabela "clientes_contratos" campo "tipoContrato"
   - Fazer query filtrada por banco:
```sql
   SELECT DISTINCT tipo_contrato 
   FROM clientes_contratos 
   WHERE banco = $1 
   ORDER BY tipo_contrato;
```

2. TORNAR DROPDOWN DINÂMICO NO FRONTEND:
   - Quando usuário seleciona um banco → disparar evento
   - Fazer requisição para buscar tipos daquele banco
   - Atualizar opções do dropdown "Tipo de Contrato"
   - Se banco estiver vazio → mostrar todos os tipos (comportamento atual)

3. CÓDIGO FRONTEND ESPERADO (exemplo):
```typescript
   const [bancoSelecionado, setBancoSelecionado] = useState("");
   const [tiposContrato, setTiposContrato] = useState([]);
   
   useEffect(() => {
     if (bancoSelecionado) {
       // Buscar tipos apenas deste banco
       fetch(`/api/tipos-contrato?banco=${bancoSelecionado}`)
         .then(res => res.json())
         .then(data => setTiposContrato(data));
     } else {
       // Se sem banco selecionado, buscar todos
       fetch('/api/tipos-contrato')
         .then(res => res.json())
         .then(data => setTiposContrato(data));
     }
   }, [bancoSelecionado]);
```
