   - Verificar no banco se salvou: SELECT * FROM ai_prompts WHERE type='roleplay' AND is_active=true;

2. GARANTIR QUE SISTEMA USA PROMPT DO BANCO:
```typescript
   // ERRADO (hardcoded):
   const systemPrompt = "Você é um cliente servidor público..."; // ❌ fixo
   
   // CORRETO (do banco):
   const prompt = await db.select()
     .from(aiPrompts)
     .where(eq(aiPrompts.type, 'roleplay'))
     .where(eq(aiPrompts.isActive, true))
     .orderBy(desc(aiPrompts.version))
     .limit(1);
   
   const systemPrompt = prompt?.promptText || DEFAULT_ROLEPLAY_PROMPT; // ✅ do banco
```

     * Prompt global (se scope='global')
```typescript
   // Buscar prompt específico da equipe OU global
   const prompt = await db.select()
     .from(aiPrompts)
     .where(eq(aiPrompts.type, 'roleplay'))

       or(
```
