Quero reformular completamente o sistema de metas e performance dos vendedores.

IMPORTANTE:

Remover totalmente qualquer uso da palavra "ranking" do sistema.

O modelo agora será baseado em NÍVEIS INDIVIDUAIS.

Deve ser escalável para novas categorias no futuro.

🎯 OBJETIVO

Implementar um sistema de metas com:

Meta Geral

Meta Cartão (paralela e independente)

Regras:

A produção de cartão soma na meta geral.

A produção geral NÃO soma na meta cartão.

Cada categoria tem seus próprios níveis e premiações.

O sistema deve calcular automaticamente o nível com base na produção acumulada do mês.

📦 1. CRIAR ESTRUTURA DE BANCO

Criar tabela:

meta_niveis

Campos:

id (PK)
categoria (VARCHAR) → 'GERAL', 'CARTAO' (deixar preparado para novas categorias)
nome_nivel (VARCHAR) → BRONZE, PRATA, OURO, DIAMANTE
valor_minimo (NUMERIC)
valor_maximo (NUMERIC)
premio (NUMERIC)
ordem (INT)

Criar índices para:

categoria

valor_minimo

valor_maximo

📦 2. LÓGICA DE CÁLCULO DE NÍVEL

Criar função:

getNivelAtual(vendedorId, categoria)

Regras:

Buscar produção acumulada do mês atual.

Se categoria = GERAL:

somar toda produção.

Se categoria = CARTAO:

somar apenas produção do tipo CARTAO.

Buscar na tabela meta_niveis o nível correspondente onde:
valor_minimo <= produção <= valor_maximo.

Retornar:

nível atual

prêmio atual

próximo nível

valor faltante para próximo nível

percentual de progresso dentro do nível
